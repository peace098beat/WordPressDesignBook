## 各固定ページの作成
 - 固定ページ名
 	 - Top
 	 	 - welcomページ
 	 - Profile
 	 	 - 崎浜光氏の紹介
 	 - Gallery
 	 	 - 崎浜光氏の作品集
 	 - Blog
 	 	 - 崎浜光氏のブログ
 	 - Show
 	 	 - イベントなどの活動記録
 	 	 - 「未来と過去のページ」

		 - 過去：ぺインターとして参加したイベントでの作品を掲載。
